# 🌐 IPTrace — IP Location Tracker

A simple web app to track the location of any IP address or domain.

---

## 🚀 Features

- 🔍 Search any IP or domain  
- 📍 Live map using Leaflet.js  
- 🌍 Shows country, city, region  
- 📡 Displays ISP & ASN info  
- 🕐 Timezone + local time  
- 🌗 Dark / Light mode toggle  
- ⚡ Auto-detect your IP  

---

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla)  
- Leaflet.js (map)  
- OpenStreetMap tiles  
- ipapi.is (IP geolocation API)
